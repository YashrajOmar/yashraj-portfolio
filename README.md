## ⚡ Setup / Local Deployment
No build tools are required. To run this portfolio locally:
1. Clone this repository:
   ```bash
   git clone https://github.com/YashrajOmar/yashraj-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd yashraj-portfolio
   ```
3. Open `index.html` in any modern web browser or use a live server extension.
